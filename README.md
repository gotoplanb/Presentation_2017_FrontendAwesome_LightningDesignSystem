## Demo

### Dependencies

* NodeJS
* Yarn

### Setup

1. Clone repo
2. Run `yarn install`

### Running

1. Run `node server.js`
2. Browse to `http://localhost:8080/`

### Continuing the tutorial

1. [Resume tutorial at step 6](https://www.lightningdesignsystem.com/platforms/heroku/)
